# Description

##  Mục tiêu 
- Hiểu được các cấu trúc mạng nơ-ron thông qua thực hành lập trình
- Sử dụng thư viện Tensorflow (Keras API)

## Framework
- Sử dụng ngôn ngữ Python, với thư viện tensorflow và keras
